# Area Calculator

A simple and interactive web application for calculating the area of different geometric shapes.

Built using:
- HTML
- CSS
- JavaScript

## 🚀 Live Demo

https://mwc-area-calculator.vercel.app/

---

## 📖 About The Project

This project was created as part of my learning journey in frontend web development, mainly focusing on JavaScript DOM manipulation and mathematical calculations.

The application allows users to calculate the area of different shapes quickly and easily through a clean user interface.

---

## 🎯 Objectives

By building this project, I aimed to:

- Practice JavaScript fundamentals
- Improve DOM manipulation skills
- Learn form handling and user input validation
- Build responsive UI layouts using CSS
- Apply mathematical formulas in programming

---

## ✨ Features

- Calculate area of different shapes
- Simple and user-friendly interface
- Instant results
- Responsive design
- Interactive calculations using JavaScript

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

Clone the repository:

git clone git@github.com:MoyiWycliffe/Area-calculator.git
